# HTML / SCSS

## Naming Convention

HTML과 SCSS 클래스명은 kebab-case를 사용한다.

```html
<header class="main-header">
  <section class="product-list">
    <button class="filter-chip"></button>
  </section>
</header>
```

---

# JavaScript

## Naming Convention

JavaScript 변수 및 함수명은 camelCase를 사용한다.

```javascript
const currentPage = 1;
const selectedFilters = [];

function renderProducts() {}
function fetchProductData() {}
```

# Responsive Design Specification

## Desktop

### Layout

- Container Max Width : 1200px
- 중앙 정렬 (margin: 0 auto)
- 최소 좌우 여백 80px 확보

### Spacing

- 카드/컴포넌트 간격 : 24px
- 섹션 간격 : 120px ~ 160px

### Typography

| 구분          | Size | Line Height |
| ------------- | ---- | ----------- |
| Main Title    | 48px | 1.2         |
| Section Title | 32px | 1.3         |
| Body          | 16px | 1.6         |

---
